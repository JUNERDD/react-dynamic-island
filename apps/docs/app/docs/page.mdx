# Introduction

A flexible, animated dynamic island component for React, powered by Framer Motion.

## Features

- Smooth transitions between different states and sizes
- Fully customizable content and animations
- Type-safe configuration with TypeScript
- Built with Framer Motion for high-performance animations
- Support for multiple presets (Call, Charging, Music, etc.)

## Installation

```bash
pnpm add react-dynamic-island motion
```

## Basic Usage

Import the component and configure your states. See the Component documentation for details.

```tsx
import DynamicIsland from 'react-dynamic-island'

// ... in your component
<DynamicIsland
  value={activeState}
  components={myComponents}
/>
```
